## Disambiguation Rules

- 超市（全聯/家樂福/Costco）買「食材、食物」-> 伙食費
- 超市買「清潔用品、日用品」-> 生活用品
- 超商（7-11/全家）-> 依購買內容判斷：熟食/便當/零食 -> 外食；飲料 -> 飲料；日用品 -> 生活用品
- 餐廳/外送/小吃攤/夜市 -> 外食
- 朋友聚餐、慶祝餐會（非日常外食性質）-> 娛樂
- 加油/停車/ETC/公共運輸/車輛維修保養 -> 交通
- 看診/掛號/處方藥 -> 醫療保健
- 維他命/保健食品/營養品 -> 醫療保健
